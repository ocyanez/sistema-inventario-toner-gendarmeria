Sistema de Inventario de Tóner

Aplicación de escritorio desarrollada durante mi práctica profesional en Gendarmería de Chile.

Debido a políticas institucionales, el código fuente no puede ser compartido. Se incluye una versión ejecutable junto con evidencia visual de activos y sistema.

---

Funcionalidades

- Gestión de inventario de tóner
- Registro de entradas y salidas
- Edición de registros
- Interfaz simple y eficiente

---

Tecnologías utilizadas

- Electron
- JavaScript
- Python
- SQLite
